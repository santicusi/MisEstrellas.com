Crea una plataforma web de fidelización llamada "MisEstrellas.com" usando HTML, CSS y JavaScript vanilla con soporte bilingüe (Español/Inglés).

# ESPECIFICACIONES TÉCNICAS
- **Stack:** HTML5, CSS3, JavaScript ES6+
- **Librerías:** Firebase SDK v9, qrcode.js, jsQR, intl-tel-input
- **Idiomas:** Español (por defecto) e Inglés con toggle switch
- **Diseño:** Sencillo, profesional, tecnológico
- **Colores:** Fondo negro (#1A1A1A), naranja (#FF8C42), blanco (#FFFFFF)
